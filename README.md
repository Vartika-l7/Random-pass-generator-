# Random-pass-generator-
A sleek, responsive web application that generates secure, randomized passwords at the click of a button. This tool ensures your passwords meet security standards by combining uppercase letters, lowercase letters, numbers, and special symbols.

# Features
One-Click Generation: Instantly creates a 12-character secure password.

Character Diversity: Guaranteed inclusion of at least one uppercase letter, lowercase letter, number, and symbol.

Modern UI: Built with a dark-themed aesthetic using the "Bricolage Grotesque" typography.
